编译原理语法规则

程序 ->
变量声明语句 |
函数声明语句
变量声明语句 -> int 标识符 (=表达式)? ;
函数声明语句 -> (int | void) 标识符(参数列表)函数语句块
参数列表 -> int 标识符(,int 标识符)*
常量 -> [0-9]+
标识符 -> [A-Za-z][A-Za-z0-9]*
运算对象 ->标识符 | 标识符(参数) | 常量
参数 -> ε | 运算对象(,运算对象)*
函数语句块 -> {(if语句块|赋值语句|表达式;|变量声明语句|while语句块)+}
赋值语句 -> 标识符 = 表达式;
if语句块 -> if(表达式)函数语句块 (else if语句块 | else 函数语句块)? //不允许无花括号的if语句
while语句块 -> while(表达式)函数语句块
表达式 -> 运算对象 | (表达式) | 表达式(+-*/%)表达式
